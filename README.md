# 动漫
整合[樱花动漫](http://www.yinghuacd.com/)、[奈飞影视](https://www.naifei.org/)资源的app。
- 主要是为了学习[Jetpack Compose](https://developer.android.com/jetpack/compose?hl=zh-cn)
- 樱花动漫爬取的网页链接，奈飞用网上找的接口
- 有些功能还没有做，慢慢更新
